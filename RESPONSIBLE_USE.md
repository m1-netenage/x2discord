# Responsible Use — スクレイピングに関する注意とガイドライン

このリポジトリで提供するコードはスクレイピング用途に使えますが、利用者は以下の点を守ってください。違法または不正な行為に対する責任は利用者自身にあります。

## 重要な注意点
- サイトの利用規約（Terms of Service）や `robots.txt` の規定を尊重してください。
- 認証が必要なデータ、不正に取得したデータ、個人情報（PII）の収集・公開は行わないでください。
- サイトに負荷を与えないように、合間をあけたリクエスト（レート制限）を必ず設定してください。

## 推奨設定（例）
- 同一ドメインへのリクエストはデフォルトで 1 秒あたり 1 回以下にする。
- リクエスト間にランダムな遅延（0.5〜2秒）を挿入する。
- 適切な `User-Agent` を設定し、可能なら連絡用のメールアドレスを含める（例: `x2discord-bot/1.0 (+you@example.com)`）。
- 大量取得が必要な場合はサイト管理者に事前に許可を取る。

## 法的・倫理的配慮
- スクレイピングが当該サイトやサービスの利用規約違反になる場合、当該行為は違法または損害賠償の対象となる可能性があります。
- 国や地域によっては不正アクセスやコンピュータ不正利用の法規制が厳しいため、必ず現地法を確認してください。

## 連絡先・問い合わせ
問題やブロックの通知を受け取るために連絡先を公開することを推奨します。連絡先をここに記載する場合は、メールアドレスや GitHub Issues の利用方法を明示してください。
